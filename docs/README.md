# AquaTrack

## Build
```
scripts/build.sh  [debug | release]
```
> If no argument is passed, `release` will be used as default


## Running on QEMU
```
scripts/qemu.sh [debug | release]
```
> If no argument is passed, `release` will be used as default


## Flash

```
scripts/flash.sh [debug | release]
```
> If no argument is passed, `release` will be used as default
